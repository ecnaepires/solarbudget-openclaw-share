# PDF Fixture Folder

This folder is intentionally empty in the public repository.

Add redacted invoice PDFs here when preparing a real extraction audit.

Suggested naming:

- `works-celesc-1.pdf`
- `fails-enel-a4-1.pdf`
- `weird-cpfl-2.pdf`

Recommended minimum set:

- 2 to 3 PDFs that work well
- 2 to 3 PDFs that fail
- 1 mixed batch from different distributors

Also include a short note or manifest entry explaining:

- distributor
- whether it should pass or fail
- what fields are wrong
- what the expected result should be

See `manifest.example.json`.
